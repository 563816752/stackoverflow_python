## 合并两个列表

怎样合并两个列表?

例如:

```python
listone = [1,2,3]
listtwo = [4,5,6]
```

我期待:

```python
mergedlist == [1, 2, 3, 4, 5, 6]
```

***

在Python中非常容易.

```python
mergedlist = listone + listtwo
```
